# Deep Learning (esercitazione)

E' raccomandato installare il progetto in un ambiente virtuale.

```bash
# Linux/MacOS/WSL
python -m venv venv_unix
source venv_unix/bin/activate

# Windows
py -m venv venv_win
venv_win\Scripts\activate

# Powershell
py -m venv venv_psh
.\.venv_psh\Scripts\Activate.ps1
```

Installare dipendenze utili.
```bash
`pip install setuptools`
```
